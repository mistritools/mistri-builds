Frequently Asked Questions

Everything you need to know about MISTRI Builds tools.

Is MISTRI Downloader free?+

Yes, completely free. No subscription fees, no hidden costs, no premium tiers. All MISTRI Builds tools are free to download and use forever.

Does it work on Mac or Linux?+

Currently MISTRI Downloader is only available for Windows 10 and Windows 11. Mac and Linux versions are not planned at this time.

What video qualities are supported?+

The following quality options are available:

  • Best available quality (auto)
  • 1080p Full HD
  • 720p HD
  • 480p Standard
  • 360p Low

Note: available quality depends on what the YouTube video actually offers.

Can I download full playlists?+

Yes! Just paste the playlist URL, click Get Info, and the app will detect all videos automatically. You can then download the entire playlist in your chosen format and quality.

Is there a file size limit?+

No file size limit is imposed by the app. You can download videos of any length. Very long videos will naturally take more time depending on your internet speed and YouTube's servers.

Do I need to install Python to use it?+

No. The app ships as a standalone .exe file. Just download and double-click to run. No Python, no extra installs, nothing.

Is it safe to use?+

Yes. The app does not collect any personal data, does not require login, and only connects to YouTube to fetch video info. We recommend always downloading from the official GitHub releases page to ensure you have the genuine file.

How do I update to the latest version?+

Visit the GitHub releases page to download the latest version. Simply replace the old .exe with the new one — no uninstallation needed.

Can I download MP3 audio only?+

Yes. Select the MP3 Audio option in the Format selector before clicking Download. The app will extract and save the audio track as an MP3 file.

Where are my downloaded files saved?+

By default files are saved to your Downloads folder. You can change the save location using the Browse button in the app before starting a download.
